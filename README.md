# 💰 Sistema Bancário em Java

Este projeto é um sistema bancário simples desenvolvido em Java, com o objetivo de praticar lógica de programação, estruturas de dados e organização de código.

## 🚀 Funcionalidades

- Criar conta
- Login de usuário
- Ver saldo
- Depositar dinheiro
- Sacar dinheiro
- Visualizar extrato de operações

## 🧠 Conceitos aplicados

- Estruturas condicionais (`if`, `switch`)
- Estruturas de repetição (`while`, `for`)
- Arrays e matrizes
- Métodos
- Validação de dados
- Entrada de dados com `Scanner`

## 📂 Estrutura do projeto

- `SistemaBanco.java` → Classe principal com toda a lógica do sistema

## ▶️ Como executar

 - Compile o arquivo:

```bash
javac SistemaBanco.java
```

Execute:

java SistemaBanco

## 📌 Observações

- O sistema utiliza arrays com limite de 100 usuários

- As senhas são armazenadas em texto simples (apenas para fins didáticos)

- O extrato é limitado a 100 operações por usuário

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em aprendizado, simulando operações básicas de um sistema bancário e preparando para evolução com Programação Orientada a Objetos.
